Rgba

public struct Rgba : IEquatable<Rgba>

Packed RGBA color. Each ushort component stores an 8-bit channel value in its low byte and one byte of a 32-bit metadata word in its high byte. This layout mirrors the Zig/TypeScript implementation exactly so that color and intent live in a single 64-bit value and can be compared with integer equality. Use the static factory methods (FromValues, FromInts, FromIndex, FromHex, …) to build values and the accessors (RedByte, GreenByte, …, Intent, Slot) to read them.
