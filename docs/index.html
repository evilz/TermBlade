TermBlade

Razor-sharp Terminal UIs for .NET 10

A cell-based rendering engine with ANSI escape sequence support, styled text, a rope-backed edit buffer with full undo/redo, Razor component hosting, and a plugin architecture.

Features

🎨

Cell-Based Rendering

2D terminal cell grid with text, borders, blitting, and alpha blending. Full ANSI escape sequence support with true-color and 256-color palettes.

📝

Rope-Backed Editor

High-performance text editing with a rope data structure. Full cursor navigation, insert/delete operations, and unlimited undo/redo.

⚡

Diff-Based Renderer

Only redraws cells that have changed between frames. Alternate screen support for clean terminal management.

🧩

Plugin Architecture

Extensible plugin system with registration and lifecycle management. Build and share custom components.

🔧

Razor Component Host

Use familiar Razor syntax to build terminal UIs. Component-based architecture with data binding and event handling.

🎯

Rich Renderables

Built-in components: Box, Text, Input, Select, TreeView, Slider, ScrollBox, Markdown, Code highlighting, Diff viewer, and more.

Quick Start

TermBlade offers two approaches to build terminal UIs. Use TermBlade.Core for low-level, imperative drawing with full control over cells, colors, and ANSI sequences. Use TermBlade.Razor for a declarative, component-based experience with familiar Razor syntax.

Option A — Core API

1 Install the package

dotnet add package TermBlade.Core

2 Draw your first UI

using TermBlade.Core.Ansi;
using TermBlade.Core.Buffer;

using var buf = CellBuffer.Create(60, 20);
var bg     = Rgba.FromInts(0,   0,   0);
var fg     = Rgba.FromInts(255, 255, 255);
var accent = Rgba.FromInts(0,   200, 255);

buf.Clear(bg);
buf.DrawBox(0, 0, 60, 20, accent, bg, BorderStyle.Rounded, BorderSides.All,
            fill: true, title: " My App ");
buf.DrawText("Hello from TermBlade!", 4, 3, fg, bg);

3 Build & run

dotnet build
dotnet run

Option B — Razor Components

1 Install the package

dotnet add package TermBlade.Razor

2 Create a component (MyApp.razor)

<Box Width="100%" Height="100%" Border="true" BorderStyle="rounded" BorderColor="#00c8ff" BackgroundColor="#000000" Title=" My App ">
    <Text Content="Hello from TermBlade!" Fg="#ffffff" />
</Box>

3 Host and run (Program.cs)

using TermBlade.Razor;

var host = Host.CreateDefaultBuilder(args)
    .UseTermBladeRazor<MyApp>()
    .Build();

await host.RunAsync();
dotnet run

Key Types

TypeDescription
RgbaPacked color: RGB, ANSI-256 indexed, or terminal-default intent
CellBuffer2D terminal cell grid with text, borders, blitting, alpha blend
TextBufferStyled text storage (read-only display)
EditBufferRope-backed editor: cursor, insert/delete, undo/redo
RendererDiff-based ANSI terminal renderer with alternate-screen support
SyntaxStyleNamed style definitions with priority-aware merging
EventEmitterSimple on/off/once/emit event bus
PluginRegistryIPlugin registration and lifecycle

Project Structure

  Ansi/                      # Rgba, AnsiCodes, TextAttributes, ColorIntent
  Buffer/                    # CellBuffer, Cell, BorderStyle
